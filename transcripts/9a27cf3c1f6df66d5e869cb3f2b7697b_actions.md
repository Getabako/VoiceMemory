# 岡本とシステム改善: 記録補助機能 - 岡本: 支援システム改修要望

### 📋 会議要約
この会議は、岡本さんが要望している支援システムの改修について話し合われました。主な内容は、新規登録ユーザーの読み込み不具合の修正、アセスメント管理機能の改善、記録保存機能の複数選択対応、活動記録の一括入力機能の追加、そして個別支援計画書のデータ移行方法の検討です。最終的には、CSV形式でのデータ移行とAIを活用した記録作成補助機能の開発に向けて進むことになりました。

### ✅ アクションリスト
**岡本**
- [ ] 個別支援計画書の様式データまたは画像を共有（名前を伏せたもの）
- [ ] 現在の業務フローで他にシステム化できる部分がないか洗い出す

**話者A**
- [ ] 新規登録ユーザーの読み込み不具合を修正
- [ ] アセスメント管理機能の一覧表示が動くように修正
- [ ] 活動内容の複数選択（チェックボックス式）に対応
- [ ] 活動記録の一括入力機能を追加（全体活動記録から児童ごとの記録作成）
- [ ] CSV形式での個別支援計画書データ取り込み機能の開発
- [ ] AIを活用した記録作成補助機能（連絡帳向け文章自動生成）の開発
- [ ] CSVファイルの取り扱い手順書を作成

### 📌 決定事項
- 記録システムを改修し、新規登録ユーザーが問題なく読み込まれるようにする。
- アセスメント管理機能（アセスメント一覧、支援計画一覧、成長記録一覧）を正常に動作するようにする。
- 活動内容を複数選択できるチェックボックス式の記録システムに改善する。
- 複数児童の活動記録を一括で入力できる機能を追加する。
- 個別支援計画書データをCSV形式で取り込めるようにする。
- AIが保護者向け連絡帳の文章を自動生成する機能を開発する。
- 岡本さんは会社経費で記録補助デバイスを購入可能。

### 💡 補足・注意点
- 記録補助デバイスは紛失しやすいため、注意が必要。
- 個別支援計画書のデータ移行は、CSV形式が最も効率的であると判断された。
- ExcelでCSVファイルを開くと文字化けする可能性があるため、注意が必要。
- スタッフは保護者への報告内容について、AIによる自動生成された文章をそのまま使用したいという要望がある。
- 記録システムは、保護者への報告内容を簡潔に記述できることが望ましい。
