# 高崎・田中: AIスライド作成と自動化 - 田中・高崎: Notebook LMとAI自動化

### 📋 会議要約
今回の会議は、AIを活用したスライド作成と自動化がテーマ。特にNotebook LMというツールに焦点を当て、クロードコードやサブエージェントの活用事例を共有。高崎氏がAIチームを編成し、スライド作成から動画生成、YouTubeアップロードまでの自動化ワークフローを構築した事例を紹介。Notebook LMと他のAIツール（ナノバナナプロ等）との連携や、今後の展望について議論された。
