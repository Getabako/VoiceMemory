# 高崎翔太: イフ塾ピッチ - 高崎正太: 岩手塾とAI新義務教育

### 📋 会議要約
この会議は、高崎翔太氏によるイフ塾ピッチに向けた準備を目的としています。主な内容は、岩手塾とAIを活用した新しい義務教育についてのプレゼンテーションに対するフィードバック、改善点の洗い出し、および今後の戦略策定です。特に、プレゼンの焦点を絞り、具体的な成果を示すこと、審査基準を意識したアプローチの重要性が議論されました。また、高崎氏の教育への熱意をより強く打ち出すべきであるという意見が出ました。

### ✅ アクションリスト
**高崎翔太**
- [ ] プレゼンの焦点を絞り、AIワークフローの具体的な成果を示すデモ（LINEミニアプリ作成など）を検討する
- [ ] 審査基準（特に10項目の採点基準）を詳細に確認し、プレゼン内容に反映させる
- [ ] 教育に対する情熱や問題意識をより強く表現する
- [ ] LINEミニアプリの担当本部長へのプレゼン機会を探る

**洋**
- [ ] カフェ個室でのLINEミニアプリ活用について、高崎氏に実験台としてのフィードバックを提供する
- [ ] 体育館・宿泊施設向けのパスワード発行システムの開発について、高崎氏に詳細を伝える

### 📌 決定事項
- 高崎氏はイフ塾ピッチで優勝を目指し、審査基準を意識したプレゼンを行う
- LINEミニアプリを活用したビジネスモデル（月額課金など）を検討する
- 体育館・宿泊施設向けのパスワード発行システムの開発を検討する

### 💡 補足・注意点
- プレゼンが散漫にならないよう、AIワークフローの説明と教育への思いを結びつける必要がある
- 審査基準を意識しつつも、高崎氏自身の熱意や個性を失わないようにバランスを取ることが重要
- 具体的なデモ（LINEミニアプリ作成など）は、ネット環境に左右される可能性があるため、事前に準備しておく必要がある
