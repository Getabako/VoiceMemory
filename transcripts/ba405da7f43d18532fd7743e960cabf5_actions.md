# AI導入と補助金・診断ツール検討 - 高崎・田中: AI研修と助成金活用

### 📋 会議要約
この会議では、AI導入支援とそれに関連する補助金・助成金の活用について議論されました。高崎氏が開発したAI導入支援サービスとそのマネタイズ方法、特に人材開発支援助成金の活用に焦点が当てられました。また、AIを活用した診断ツールや申請書類作成支援機能、LINE連携など、具体的なサービス内容が紹介されました。さらに、補助金申請プロとの連携や今後の展開についても話し合われました。

### ✅ アクションリスト
**高崎**
- [ ] AIフェスに掲載する診断ツールを選定する（より面白い診断ツール）
- [ ] 厚生労働省の電子申請を社長に試してもらう
- [ ] 毎月定例のセミナー開催を検討する

**田中**
- [ ] 補助金助成金のプロの人と連携する
- [ ] 高崎氏のサービスページのURLを共有する

### 📌 決定事項
- AI導入支援サービスと人材開発支援助成金を組み合わせたパッケージを販売する
- 補助金申請の書類作成をAIで支援するツールを提供する

### 💡 補足・注意点
- 補助金申請のオンライン手続きをAIで自動化することのグレーゾーン（GビズIDの扱い）
- 申請手続きが煩雑である現状に対する問題提起
- 過去のゲームを彷彿とさせるAIキャラクターによるUIの面白さ
