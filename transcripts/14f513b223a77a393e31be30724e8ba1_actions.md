# システム改修MTG(高崎・辻江) - システム改善MTG（高崎・辻江）

### 📋 会議要約
この会議は、システム改修に関する高崎さんと辻江さんの間で実施された。主な内容は以下の通り。
1.  現在導入中のシステムの課題点（端末ごとの登録問題、一覧表示の不具合）の洗い出しと修正依頼
2.  記録補助機能の改善（複数選択可能な活動内容、児童ごとの記録作成）
3.  個別支援計画書のデータ連携方法の検討（CSV形式でのインポート）
4.  保護者への情報伝達方法の見直し（連絡帳の自動生成）
5.  その他、特性シートの項目削減や勉強会のテーマに関する議論

### 📌 決定事項
- 活動内容は複数選択できるチェックボックス式に変更
- 記録は全体活動を選択後、児童ごとにメモを取る形式にする
- 個別支援計画書はCSV形式でインポートできるようにする
- 記録作成時に保護者への連絡事項を自動生成する機能を検討
- 特性シートの質問項目を精査し、保護者の負担を軽減する
- 次回の勉強会テーマはカウンセリングとする

### 💡 補足・注意点
- 高崎さんから、顔認証と声認証を用いた児童の状態記録システムと、ウェアラブル端末による運動データ記録システムについての構想が共有された。
- チェックイン時の作業が増えることによる現場の負担増が懸念されている。
