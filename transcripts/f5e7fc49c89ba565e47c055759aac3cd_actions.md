# 授業発表会ふりかえり/今後 - 鈴木ら: 発表会ふりかえり/荒らし対策

### 📋 会議要約
この会議は、授業発表会の振り返りと今後の展開、特に荒らし対策について議論されました。また、ジェンスパーク接続状況やアンチグラビティのブラウザ機能の問題点などが共有されました。後半では、伊吹塾の集客に繋がるようなビジネス展開として、AIを活用したサービス提供を検討し、そのためのウェブサイト改善策について話し合われました。第三者からの評価を得て改善を進める方針が決定されました。

### ✅ アクションリスト
**匠先輩**
- [ ] ウェブサイトの中身を整理する
**全員**
- [ ] ウェブサイトのリンクをナルテック連絡用に共有する
- [ ] ウェブサイトに対する意見をまとめる
- [ ] 意見を元にウェブサイトの内容を精査する
**誰か**
- [ ] 情報発信するようなノートの自動ブログを作る
- [ ] Instagramの自動投稿を作る

### 📌 決定事項
- 荒らし行為に対しては、制裁（奈落の底に飛ばす等）を検討するが、まずは注意喚起を行う
- ウェブサイトを第三者に評価してもらい、意見を元に内容を精査する
- 伊吹塾のブログをノートで運用することを検討する

### 💡 補足・注意点
- 男子グループの発表への主体性の低さと、発表中の荒らし行為が問題点として挙げられた
- ワールドガードの導入は管理が大変なため、慎重に検討する必要がある
- AIを活用したサービス提供は魅力的だが、ウェブサイトが機械的で怪しく見えるため、実績や事例、リアルな情報を追加して信頼性を高める必要がある
- AIによる発信は冷めた印象を与える可能性があるため、人間が関わる部分も重要
