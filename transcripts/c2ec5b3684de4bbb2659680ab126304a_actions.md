# 2026-03-13 08:49:42 - 高崎翔太: ゲームで秋田を活性化する提案

### 📋 会議要約
高崎翔太氏による、ゲーム開発を活用して秋田県の経済活性化と不登校児支援を目指す「北綱 未来過ぎの守護者」プロジェクトの提案。人口減少と不登校児増加という課題に対し、ゲームをハブとして、ECショップによる特産品販売、AIナビゲーターによる観光支援、ユーザー参加型ゲーム開発を組み合わせた新しい教育・経済モデルを提示。北東銀行に対し、ゲーム内ECの実現に向けた協力（企業マッチング・支援）を要請。

### ✅ アクションリスト
**高崎翔太**
- [ ] 北東銀行との連携による、ゲーム内ECのパイロット事業（5社、月30万円の売上目標）の実現
- [ ] 秋田の特産品とコラボレーションしたキャラクターの商品企画に子供たちを参加させる

**北東銀行**
- [ ] 高崎氏の提案するゲーム内ECの実現に向けた、取引先とのマッチングや支援を検討

### 📌 決定事項
- ゲーム「北綱」を完全無料のハブとしてリリースし、認知度を最大化する
- キャラクターの商用利用を無償解放し、秋田の特産品とのコラボレーションを促進する

### 💡 補足・注意点
- ゲームを未完成でリリースし、ユーザーと共に開発を進める
- AI事業による安定収益と岐阜塾の人材育成ノウハウを基盤に、持続可能なエコシステムを構築する
- 地元企業との連携実績（3件の案件受注、累計受注額約90万円）がある
