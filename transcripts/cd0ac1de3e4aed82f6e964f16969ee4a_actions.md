# 2026-04-03 10:59:17 - 先生向け:授業でAI活用のススメ

### 📋 会議要約
この会議は、学校の先生向けに授業でのAI活用のススメをテーマに行われた。AIを業務効率化や生徒の学習支援に活用する方法、具体的なAIツール（ChatGPT、Gemini、Claude）の紹介、著作権に関する注意点などが議論された。先生がAIに慣れ親しみ、生徒に適切な利用方法を教えることの重要性が強調された。

### ✅ アクションリスト
**久先生**
- [ ] Geminiを使って、生徒の指導要録作成を効率化する。

**参加者**
- [ ] Geminiアプリをインストールし、パソコンで開いてすぐに使える状態にする。
- [ ] 1日1回、AIに質問してみる。
- [ ] AIを使ったエピソードを生徒の前で話す。

**稲さん**
- [ ] Facebookコミュニティ等でAI講座を継続する。
- [ ] 参加者のAIに関する質問に対応する。

### 📌 決定事項
- 先生方がAIを活用し、生徒にAIの正しい使い方を教える。
- 特に、Geminiを学校の先生が活用しやすいツールとして推奨する。
- AIの活用事例として、教材作成や生徒の個別指導への応用を検討する。

### 💡 補足・注意点
- 無料版AIは機能が限定的なため、教育利用には有料版が望ましい。
- AIによる丸写しは避け、壁打ちによるクオリティ向上を目指す。
- 画像生成AIの著作権には注意が必要（特に商業利用）。
