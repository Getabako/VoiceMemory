# 2026-04-15 06:26:28 - 田中・高崎: クロードコード活用戦略

### 📋 会議要約
この会議は、クロードコード実践会が2000名突破したことを記念し、高崎氏をゲストに迎えてクロードコードの活用戦略について議論されました。セミナーの内容や形式、初心者から中級者へのステップアップ、具体的なスキル習得、自動化の定義などについて意見交換が行われました。実践的なワークフローの構築や、参加者のレベルに合わせた講座の実施などが提案されました。

### ✅ アクションリスト
**田中**
- [ ] クロードコードを活用したセミナーの企画・運営
- [ ] 参加者のレベルに合わせたセミナーのコンテンツ作成（初心者向け、中級者向けなど）

**高崎**
- [ ] クロードコードセミナーで話す内容を検討（アンチグラビティとの違い、実践的な活用事例など）
- [ ] クロードコードを活用したコンテンツ作成、配信事例の紹介

**山崎**
- [ ] 大学の授業関連業務の自動化事例を紹介
- [ ] クロードコードでできること、つまづきやすいポイントなどを共有

### 📌 決定事項
- クロードコード実践会向けに高崎氏によるセミナーを開催
- セミナーは参加者のレベルに合わせて、講座形式と実践形式を組み合わせる
- 初心者向けには、安全で課金の少ないスキル作成から始める
- 中級者向けには、よりマニアックな講義と黙々会形式での実践を行う

### 💡 補足・注意点
- 画像生成、動画生成は課金が高額になる可能性があるため、注意が必要
- 自動化の定義は人によって異なるため、セミナー内で明確にする必要がある
- スキャナーの性能が低い場合、クロードコードでのOCR精度向上は難しい可能性がある
- ノートへの投稿はAPIがないため、難易度が高い
