# 匠塾: 和崎匠の漫画化計画 - 匠・五岡外・田中: 漫画出版企画

### 📋 会議要約
この会議は、和崎匠の過去を題材とした漫画出版企画の立ち上げを目的としています。参加者は、匠塾長である和崎匠本人、漫画制作担当の五岡外、原作者の田中、そして高崎支援歴が長い小林正宗です。会議では、漫画のターゲット層、作風、具体的なストーリー構成、ページ数などが議論されました。出版に向けて、関係者全員が得をするような形を目指し、実績作りやスキルアップに繋げることを目標としています。

### ✅ アクションリスト
**五岡外**
- [ ] 第1話のページ数、内容について和崎匠と相談する
- [ ] 漫画のキャラクターデザイン（等身、デフォルメ具合）を検討する

**和崎匠**
- [ ] 過去の経験から、漫画に盛り込むエピソードを選定する
- [ ] Kindleで出版した本のPDFファイルを共有する

### 📌 決定事項
- 漫画のターゲット層は、発達障害を持つ子供を抱える保護者や支援者とする
- 漫画の作風は、事実を元にしたフィクションとし、場所はオリジナルでも良いが、内容はノンフィクションに近い形とする
- 第1話は和崎匠の紹介編とし、幼少期は軽く、小学校後半から中学校時代を中心に描く

### 💡 補足・注意点
- 和崎匠の中学校時代のエピソードは衝撃的な内容も含まれる可能性があるため、表現方法（デフォルメなど）を工夫する必要がある
- 漫画制作にあたり、五岡外の負担を考慮し、柔軟に対応する
