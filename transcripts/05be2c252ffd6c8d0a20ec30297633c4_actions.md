# 高崎と動画AI:Claude+Remotion自動化 - 高崎・朝ライブ: AI動画制作の自動化

### 📋 会議要約
この会議は、AI動画制作の自動化をテーマに、特にClaudeとRemotionを活用した最新情報や、その過程で直面した課題について議論されました。高崎氏の経験を基に、自作ボイスの作成や、ジェンスパークを利用したSNS投稿の自動化など、具体的な取り組みが紹介されました。また、動画コンテンツの価値や、有料サービスに頼らない自己完結型のシステム構築の重要性についても意見が交わされました。

### 💡 補足・注意点
- 自作ボイスの作成には、録音環境の品質が重要
- ジェンスパークのAI画像生成は、最近クオリティが低下している可能性あり
- 有料の外部サービスを利用すると、ワークフローの繋がりが不自然になる場合があるため、可能な限り自作で代替する
- GitHub Actionsを利用することで、日々のタスクを自動化できる
