# 2026-03-31 15:59:45 - ナレテック:マイクラサーバと鈴木君の件

### 📋 会議要約
この会議では、ナレテックのマイクラサーバーの今後の扱い、生徒である鈴木君へのプログラミング指導の可能性、そしてIF塾のサイト掲載内容について議論されました。マイクラサーバーのデータ保管方法、鈴木君への体験授業の提案、サイトの料金プランの見直しなどが話し合われました。

### ✅ アクションリスト
**ナレテック**
- [ ] マイクラサーバーのワールドデータをローカルに保存する
**IF塾**
- [ ] 鈴木君の親御さんに連絡を取り、無料相談と体験授業を提案する
- [ ] GPTz研究会のライブの様子を鈴木君の親御さんに共有する
- [ ] IF塾のサイトの料金プラン（特に黙々開発プラン）の見直しと、ジェミニへの相談

### 📌 決定事項
- ナレテックのマイクラサーバーの契約は一旦停止し、ワールドデータのみを保管する
- 鈴木君に対し、IF塾で無料相談と体験授業を提供する
- IF塾のサイトに掲載されている料金プラン（特に黙々開発プラン）について、ジェミニに相談し見直しを図る

### 💡 補足・注意点
- IF塾のホームページが「うさん臭い」という意見が出たため、サイトの見直しが必要
- 鈴木君は不登校であるため、プログラミング指導だけでなく、メンタル面のサポートも視野に入れる必要がある
- IF塾の料金設定は、客層のフィルターとしての役割も考慮する必要がある
