# 議事録: 美脚プログラムのAI活用に関するキックオフミーティング

### 👥 参加者
- 園田 様 (美脚セルフデザインプログラム 主宰)
- 上村 様 (美脚セルフデザインプログラム 事業パートナー)
- 田中 様 (AI活用コンサルタント / 技術アドバイザー)

### 📅 日時・場所・コンテキスト
- **日時**: 2026年6月5日 09:30〜
- **場所**: オンライン
- **目的**: 園田様・上村様が運営する「美脚セルフデザインプログラム」において、受講生のビフォーアフター写真の分析・フィードバック作成業務をAIで自動化するための技術的な可能性とアプローチについて協議する。

### 📋 会議要約
本会議は、園田様と上村様が運営する「美脚セルフデザインプログラム」の事業拡大に伴い、ボトルネックとなっている写真分析業務（1件あたり30分～1時間）の効率化と品質均一化を目的として行われた。現状、イベントごとに約150名が成約するなど事業は好調である一方、手作業での分析が限界に達している課題が共有された。

議論の中心は、受講生のビフォーアフター写真から体の歪みや変化をAIでどう分析するかであった。田中様より、単純な画像認識だけでは微妙な筋肉の盛り上がり等の判別が難しい可能性が指摘された。解決策として、**①AIが画像から関節点（頭、肩、くるぶし等）を自動検出し、②それらを結ぶ線を描画、③線の角度や長さを測定・比較する**という2段階のアプローチが提案された。技術的にはGeminiやGPT-4の画像認識機能の活用が有望視された。

結論として、AIによる自動化は技術的に実現可能との見通しが立った。次のステップとして、AIに分析させるための**独自の評価基準（何をもって体の変化とするか）を園田様・上村様側で明確に言語化・定義する**ことが最優先事項であると合意した。

### ✅ アクションリスト

**[園田様]**
- [ ] 独自の写真分析メソッドにおける「評価観点」を言語化し、リストアップする。（例：足首の角度、膝の向き、骨盤の傾きなど、どの部位を、何を基準に、どう判断しているかのロジックを整理）
- [ ] 過去に使用を中止した姿勢解析ソフト「スポティップ(Spotip)」について、具体的にどの分析観点が自社メソッドと合わなかったのかを文書化し、田中様に共有する。

**[上村様]**
- [ ] 園田様と協力し、分析観点の言語化・体系化をサポートする。コンサルティングの知見を活かし、ロジックツリーなどで構造化を試みる。
- [ ] 本AI導入プロジェクトの目的、ゴール、スコープ（どこまでを自動化するか）をまとめた簡易的な企画概要書を作成する。

**[田中様]**
- [ ] 共有されたビフォーアフター写真のサンプルを使い、GeminiやGPT-4の画像認識機能で関節点の自動検出が可能か、技術的な事前検証（PoC）を実施する。
- [ ] 姿勢解析AIの開発アプローチ案（マーカー方式など）と、それに必要なデータ（写真の枚数、アノテーションの要否など）について、次回の打ち合わせに向けた簡単な提案資料を作成する。

### 📌 決定事項
- 写真分析業務の自動化に向けて、AI開発プロジェクトを前向きに進めることで合意した。
- 開発の基本アプローチとして、**AIが画像から関節点を自動検出し、それらを結んだ線の角度や長さを測定・比較する**方式を第一候補として検討を進める。
- プロジェクト着手の最優先タスクとして、園田様・上村様側で、**独自メソッドの分析観点（評価基準）を明確に言語化・定義する**こととした。

### 💡 補足・注意点
- **静止画の限界**: ユーザーが撮影時に意図的に良い姿勢を取ってしまう可能性がある。AIが分析するデータ（写真）の信頼性をどう担保するかが今後の課題として認識された。
- **AIの判断基準の重要性**: 過去に筑波大学開発の姿勢解析ソフト「スポティップ(Spotip)」を試した際、分析の観点が合わなかった経緯がある。開発するAIには園田様の観点を正確に反映させることが極めて重要であり、評価基準のすり合わせを慎重に行う必要がある。
- **分析の複雑性**: 関節点の位置や角度だけでなく、筋肉の微妙な張りや体のラインといった、線だけでは表現しきれない要素をどう扱うかは、今後の追加検討事項とする。

### 🌱 ネクストステップ候補
- **次回会議の設定**: 園田様・上村様で言語化した「分析観点リスト」を基に、AIの具体的な要件定義を行うためのミーティングを2週間後を目処に設定する。
- **技術検証の共有**: 田中様による技術検証（PoC）の結果を共有し、開発の実現性や想定される精度について議論する場を設ける。
- **サンプルデータ準備**: 少数の写真サンプル（10～20枚程度）に対し、園田様が手動で「正解」となる分析コメント（アノテーション）を付与し、AIの学習データおよび評価データとして活用する準備を進める。
