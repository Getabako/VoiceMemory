# おだしょー、ちょまど: Geminiゲーム開発 - 拓磨とジェミニでゲーム開発

### 📋 会議要約
この会議は、Geminiを使用して初めてのゲーム開発を行うことを目的としています。参加者は、主に高崎先生と拓磨君で、簡単なアクションゲームを作成し、AIを使ってカスタマイズすることを目指します。会議では、開発環境の準備、キャラクターの作成、ゲームロジックの実装について議論されました。

### ✅ アクションリスト
**拓磨君**
- [ ] Geminiでキャラクターの画像を作成する
- [ ] 作成した画像を「player.png」として保存し、指定のフォルダに格納する
- [ ] Geminiで生成されたHTMLとJavaScriptのコードをコピーし、「index.html」ファイルに貼り付ける
- [ ] 必要に応じて、高崎先生の指示に従い、ファイル名や保存形式を修正する

### 📌 決定事項
- ゲームのジャンルはアクションゲームとする
- プレイヤーキャラクターは、Geminiで生成された画像を使用する
- ゲームのコードは、Geminiによって生成されたHTMLとJavaScriptを使用する
- ファイル名は「index.html」と「player.png」を使用する

### 💡 補足・注意点
- Geminiの無料版では、画像の生成回数に制限がある可能性がある
- Windowsのメモ帳でHTMLファイルを作成する際、拡張子が正しく保存されない場合がある
- 拓磨君は、ファイルの保存場所や名前の変更に苦労しているため、高崎先生が画面共有を通じて丁寧にサポートしている
