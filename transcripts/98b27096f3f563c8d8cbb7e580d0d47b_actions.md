### 📋 会議要約
本会議は、2026年5月20日水曜日午前6時半から開催された「GPTs研究会朝ライブ」の一環として行われました。主な議題は、Google I/Oで発表された最新情報（Flash 3.5、Antigravity 2.0、Gemini CLIからAntigravity CLIへの変更、Gemini Sparkなど）の共有と、同日20時に開催される「Cloud Code Agent実践会」に関する詳細な打ち合わせでした。

Google I/Oの発表内容については、Flash 3.5の速度と安価さへの期待、Antigravity 2.0の改善点、そしてGoogle製AI（特にGemini）の既存の動作不安定性やアカウント作成の厳しさに対する懸念が共有されました。特に、Androidアプリのダイレクトビルドの可能性や、CLIツールの名称変更に伴うAPI書き換えなどの影響が議論されました。

「Cloud Code Agent実践会」については、申し込み方法（専用ページからの登録）、リアルタイム参加は無料、アーカイブは3000円またはシェアで無料となること、そしてオープンチャットに300名以上が参加している状況が共有されました。また、プログラミング学習における環境構築の難しさ（初期設定、エラー時の対応、パス設定、Claudeの起動問題など）が参加者の大きなハードルとなることが認識され、そこに対する手厚いサポート体制（マニュアル、チャットでの相談）の重要性が強調されました。実践会ではAPIの話は行わず、GASを中心に進める方針が示されました。

### 💡 補足・注意点
- Google I/Oで発表された新機能（Flash 3.5、Antigravity 2.0、Gemini Sparkなど）の具体的な性能向上や利用方法は、まだ全容が明らかになっておらず、今後の検証が必要である。
- Google製AI（特にGemini全般、Antigravity）には動作が不安定（固まる、クラッシュする、指示通りに画像を生成しないなど）という既存の懸念があり、Antigravity 2.0でこれらの問題が解消されているか注目される。
- Googleのアカウント作成ルールが厳格化しており、アカウントがバンされる事象が発生しているため、利用には注意が必要である。
- Gemini Sparkと既存の「Spark」という名称が類似しており、混乱を招く可能性がある。
- 「Cloud Code Agent実践会」における初期の環境構築と設定は、参加者が最も脱落しやすい難関である。特にパス設定の不備、Claudeの起動問題、エラー発生時の対応が課題となる。
- 実践会でのサポート範囲は、物理的な問題（メモリ不足、ディスク容量不足、OS自体（Chrome OSなど）の互換性、アプリの未インストールなど）は対象外となる可能性がある。
